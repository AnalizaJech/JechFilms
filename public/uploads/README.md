# Carpeta de uploads

Esta carpeta almacena las imágenes subidas por los usuarios y administradores:

- `/posters/` - Portadas de películas y series
- `/backdrops/` - Imágenes de fondo
- `/avatars/` - Avatares de usuarios
- `/thumbnails/` - Miniaturas de episodios

## Permisos

Asegúrate de que el servidor web tenga permisos de escritura:

```bash
chmod -R 755 uploads/
```
