# Carpeta de Media

Aquí se almacenan los archivos de video:

- `/movies/` - Películas (archivos .mp4, .webm, .mkv)
- `/series/` - Series organizadas por carpetas
- `/vault/` - Contenido de caja fuerte

## Estructura recomendada para series

```text
/series/
├── breaking-bad/
│   ├── s01e01.mp4
│   ├── s01e02.mp4
│   └── ...
└── the-office/
    ├── s01e01.mp4
    └── ...
```

## Formatos soportados

- MP4 (recomendado)
- WebM
- OGG
